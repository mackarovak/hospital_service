# MedCat MVP Lite — задачи для разработки

## Описание

Это урезанная версия MVP проекта **MedCat** для первого этапа разработки.

Главная цель первого этапа — сделать минимально рабочую систему:

- пациент авторизируется по логину и паролю;
- пациент смотрит свою медкарту;
- пациент заполняет и редактирует свои личные данные;
- врач авторизируется по логину и паролю;
- врач видит своих пациентов;
- врач открывает медкарту пациента;
- врач добавляет и редактирует записи в медкарте.

---

## Что НЕ входит в первый этап

На первом этапе не делаем:

- админ-панель;
- управление ролями через интерфейс;
- аудит как отдельный модуль;
- электронную подпись;
- документы/PDF;
- назначения как отдельный сложный модуль;
- диагнозы как отдельный справочник;
- лабораторные анализы;
- интеграции с внешними системами;
- телемедицину;
- сложный RBAC;
- регистрацию пользователей через интерфейс.
